# Sobre o projeto

Este projeto foi desenvolvido em dupla e representa uma aplicação web full stack integrada, conectando front-end e back-end em um único sistema funcional.

A aplicação permite a comunicação completa entre cliente e servidor, garantindo o fluxo de interação do usuário, processamento das regras de negócio na API e retorno de dados em tempo real para a interface.

O resultado é um sistema funcional de ponta a ponta, validando a integração entre as camadas da aplicação.

## Tecnologias e versões

### Front-end
- **Angular:** `16.2.1.`
- **TypeScript**
- **Node.js**
- **HTML5**
- **SCSS**
- **JavaScript**

### Back-end
- **Java:** `17`
- **Spring Boot**
- **Maven**

## Objetivo

Construir uma aplicação web completa aplicando conceitos de arquitetura cliente-servidor, integração entre front-end e back-end, organização de código e boas práticas de desenvolvimento full stack.

Projeto desenvolvido para a disciplina **DesWeb II – Desenvolvimento Web II**, do curso de **Sistemas de Informação (SI)**.
